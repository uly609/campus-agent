# Demo Script

1. Start services with `docker compose up --build -d`.
2. Seed data with `make seed`.
3. Open `http://localhost:5173`.
4. Refresh the post feed.
5. In AI Assistant, ask `图书馆今天几点关门？` and show citations.
6. In Smart Search, search `南门 捡到 校园卡` and show retrieval explanations.
7. In Post Assistant, generate a synthetic lost-and-found draft, edit it, and confirm it.
8. Ask chat to remember `记住我喜欢图书馆靠窗座位`; open Memory Management and delete the memory.
9. Run Eval Dashboard.
10. Open Grafana and Prometheus metrics.
